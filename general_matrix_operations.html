Trace Function Definition

The Trace function is the sum of the diagonal terms of a square matrix A of order (n X n)

\begin{equation} tr A = \displaystyle\sum_{i=1}^{n} a_i \end{equation}

C# Implementation of Trace Function

The following shows the C# implementation of the Trace function for the SquareRealMatrix class. 

          
            public double Trace()
            {
              double ret = 0;
              for (int i = 0; i < this.Rows; i++)
              {
                ret += this[i, i];
              }
              return ret;
           }

Vec Operator Definition

The Vec operator of matrix A of order (n X n) is defined as:

\begin{equation} \begin{aligned}&Vec A = \begin{bmatrix}A_{.1}\\A_{.2}\\ \vdots\\A_{.n}\\\end{bmatrix} \end{aligned} \end{equation}

Where we use the column accessor $$ A_{.n}$$
C# Implementation of Vec Operator

The following shows the C# implementation of the Vec operator method for the SquareRealMatrix class. 

     
      public RealVector Vec(string MatrixName = "A")
      {
          RealVector ret = new RealVector(); //create return vector
          string MN = (this.Name == string.Empty || this.Name == null) ? MatrixName : this.Name;
          for(int i = 0; i < this.Columns; i++)
          {
              RealVector rv = this["." + (i + 1).ToString()]; //use column accessor
              ret.AddRange(rv); //ad to return vector
          }

          ret.FullRep = @"Vec\;" + MN + " = " + ret.ToLatex();
          return ret;
      }
C# Example usage of the Vec Operator method
     
      RealFactory rf = new RealFactory(); //create a real factory

      SquareRealMatrix A = rf[2, 2, //create a 2 X 2 real matrix
      1, 2,
      3, 4 
      ];

      RealVector VecA = A.Vec("A"); //use Vec operator giving the matrix a name
      StringBuilder sb = new StringBuilder();//Start building latex
      sb.Append(@"\begin{aligned}");
      
      sb.AppendFormat(@"&A = {0}", A.ToLatex() + @" \\ \\");//display A matrix
      sb.AppendFormat(@"&{0}", VecA.ToLatex("F")); //display Vec A
      sb.Append(@"\end{aligned}");

      HtmlOutputMethods.WriteLatexEqToHtmlAndLaunch(sb.ToString(), "Test_VecOperator.html"); //display Latex via mathjax
